   ## Installation
   Clone repo, `npm i` , then `npm server.js`. After starting the server navigate to localhost:3000 in an internet browser or view the live application at the heroku link below.
   
   ## Usage
   This application shows the use of PWA's in web development and also the use of indexedDB and Cache.
